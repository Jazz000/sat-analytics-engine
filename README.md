# 📈 SAT Analytics Engine

## Overview
Success on the Digital SAT requires more than just practice; it requires data-driven strategy. I built this tool to move beyond "blind studying" by quantifying my performance and identifying the specific domains where I am losing the most points.

## How it Works
1. **Error Categorization:** As I review practice tests, I log each mistake into the engine.
2. **Frequency Analysis:** The script calculates the percentage of total errors represented by each category.
3. **Study Prioritization:** The engine outputs a ranked list, ensuring my next study session focuses on my "Weakest Links" first.

## Technical Implementation
* **Language:** Python
* **Data Structure:** Utilizes Dictionaries for efficient category tracking.
* **Logic:** Implements sorting algorithms to rank study priorities dynamically.
